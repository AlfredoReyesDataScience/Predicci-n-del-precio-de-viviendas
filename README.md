## Dataset

California Housing Dataset o Ames Housing

Variables: número de habitaciones, metros cuadrados, antigüedad, ubicación, etc.

## Qué se analizó

Relación entre características y el precio

Correlaciones entre variables

Distribución de precios y outliers

## Técnicas utilizadas

Limpieza de datos (valores nulos, outliers)

EDA con histogramas, boxplots y heatmaps

Regresión Lineal y Regresión Ridge

Normalización de variables

 ## Resultados principales

Variables como tamaño y ubicación fueron las más influyentes

El modelo explicó gran parte de la variabilidad del precio (R² aceptable)

Ridge redujo el sobreajuste
## Qué aprendí

Interpretación de coeficientes

Importancia de la multicolinealidad

Cómo evaluar modelos de regresión con MAE y R²
